Designed to work for 3.3.5 WoW version (WoTLK) but may work in others if .toc file has the right WoW version number

open MultiFollow.lua for more details
------------------------------------------------------------------------
Raid or Party Chat Commands:
mf
mfstop
mount

mf = follows sender
mfstop = stop following, stop following + dismount
mount = follow sender + then mount (may have to type or press macro 2x)
------------------------------------------------------------------------
In-Game /slash commands:
/mf

/mf = will print information that addon has loaded and list commands usable in party + raid chat
